# Weather Dashboard

Starter code for the React + Tailwind weather app 
## Setup

1. Install dependencies:

```bash
npm install
```

2. Create a `.env` file from `.env.example` and add your OpenWeatherMap API key:

```env
VITE_OPENWEATHER_API_KEY=your_real_api_key_here
```

3. Start the dev server:

```bash
npm run dev
```

## Deployment

Update `vite.config.js` and `package.json` if your GitHub repo name or username differ from the defaults in this starter.
